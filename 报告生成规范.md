## 一、生成流程（铁律）

### 1.1 日期确定与数据时效策略（生成前必须执行）

```
步骤1：读取 env 中的 Today's date，确认为 T 日（如 2026-05-25）
步骤2：文件名日期 = T 日（如 20260525）
步骤3：header 显示 T 日日期
```

**⚠️ 数据时效策略（严格执行，按Tab区分）**：

| Tab              | 数据时效策略 | 说明                                      |
| ---------------- | ------ | --------------------------------------- |
| **Tab 0 要点速览**   | 当下最新   | 搜索最新新闻/事件，不限定日期                         |
| **Tab 1 国内外新闻**  | 当下最新   | 搜索最新国内外新闻，取生成时刻前最新报道                    |
| **Tab 2 AI前沿**   | 当下最新   | 搜索最新AI/科技动态，取生成时刻前最新报道                  |
| **Tab 3 全球市场**   | 最近收盘日  | A股/港股/美股/亚太/欧洲/大宗/加密/汇率债券，均取最近一个交易日收盘数据 |
| **Tab 4 价值投资风向** | 当下最新   | 机构观点、板块估值取最新可得数据                        |
| **Tab 5 关注标的**   | 最近收盘日  | 股价/涨跌幅取最近交易日收盘数据，基本面取最新可得               |
| **Tab 6 理财话题**   | 当下最新   | 存款利率、理财收益率、社区话题取最新可得数据                  |
| **Tab 7 今日总结**   | 当下最新   | 综合分析取最新可得数据                             |

**搜索关键词策略**：

- **全球市场数据（Tab 3/5 股价部分）**：使用最近收盘日日期搜索，如"2026年5月22日 A股收盘"
- **其他所有 Tab**：不限定具体日期，搜索最新信息，如"最新央行MLF操作""AI大模型最新进展"

**全球市场收盘日特殊场景**：
| 场景 | 收盘数据引用 |
|------|------------|
| 正常交易日（周一至周五） | T-1 日收盘 |
| 周一生成报告 | 上周五（T-3）收盘 |
| 周末生成报告 | 最近交易日收盘，标注"周末休市" |
| 假期中生成报告 | 最近交易日收盘，标注"假期休市" |

| 轮次       | 必搜内容                                                                                                                                                                         | 用于Tab   |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------- |

**搜索原则**：

- 批量搜索：多个相关主题合并为一次搜索
- 优先权威来源：新华社、财联社、每经AI、新浪财经
- **全球市场数据（Tab 3/5 股价部分）**：搜索时使用最近收盘日日期关键词，如"2026年5月22日 A股收盘"
- **其他所有 Tab（0/1/2/4/6/7）**：搜索时不限定具体日期，取生成报告当下最新可得信息
- 搜索不到的数据：写"暂无数据"或引用最近可得日期并标注

### 1.3 撰写报告（基于template.html）⚠️核心流程

**⚠️ 铁律：必须基于 template.html 复制后填充，严禁从零生成HTML文件！**

**操作步骤**：

```
步骤1：复制模板（必须用命令复制，不能手动重写）
  cp template.html 老盛早知道_YYYYMMDD.html

步骤2：读取复制后的文件，确认文件结构完整（8个tab-panel、CSS/JS未损坏）

步骤3：根据获取的内容更新各个tab的内容
  ** 内容丰富度要求（严格执行）**：
  - 禁止简单描述，更新内容必须丰富，至少包含100字以上，提供背景信息、市场影响、投资逻辑
  - 下面4个地方的每个占位符的描述内容必须更丰富（120字以上）：一 · 机构价值投资观点、三 · 高股息板块深度分析、社区热门话题、今日操作建议
  - 社区热门话题 的格式要求 参考 格式规范

  ** 重点突出展示要求（严格执行）**：
  - 文字内容重点必须通过 **颜色高亮** 突出，不改变字体大小
  - 上涨/利好/正面关键词 → 红色 (`#f85149`)
  - 下跌/利空/负面关键词 → 绿色 (`#3fb950`)
  - 例如："<span style=\"color:#f85149;font-weight:700;\">大涨2.84%</span>创历史新高"

 ** 替换操作要求（严格执行）**：
  - 禁止删除、新增、修改任何HTML标签
  - 禁止修改CSS样式
  - 禁止修改JavaScript代码
  - 只允许修改占位符中的文本内容

步骤4：根据真实涨跌数据，调整全球市场卡片的 class 和边框颜色
  - 上涨 → class="up"（红色）
  - 下跌 → class="down"（绿色）
  - 持平/无数据 → class="neutral"（灰色）
  - 卡片边框颜色同步调整
```

**占位符替换规则**：

| 占位符类型   | 替换方式                                                             | 示例                                                                             |
| ------- | ---------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| 纯数据类    | 直接替换为数值                                                          | `{{A股_上证指数_数据}}` → `4112.90 +0.87%`                                            |
| 文字描述类   | 替换为详细描述，特别是如下4块内容需要特别详细：一 · 机构价值投资观点、三 · 高股息板块深度分析、社区热门话题、今日操作建议 | `{{新闻1_正文}}` → 具体新闻内容                                                          |
| 涨跌class | 根据真实数据选择up/down/neutral                                          | `class="neutral"` → `class="up"`                                               |
| 卡片边框    | 根据整体涨跌调整颜色                                                       | `border-top:2px solid rgba(255,255,255,0.12)` → `border-top:2px solid #f85149` |

**禁止行为**：

- ❌ 严禁从零生成HTML文件（必须cp template.html）
- ❌ **严禁用 Write 工具一次性写入整个文件**（会把模板的 CSS/JS 结构覆盖掉，导致文件损坏）
- ❌ 严禁修改HTML标签结构（只替换占位符内容）
- ❌ 严禁修改CSS样式代码
- ❌ 严禁修改JavaScript代码
- ❌ 严禁删除或新增占位符（如需调整请先修改template.html）

**⚠️ 填充方式铁律（严格执行）**：

```
正确做法：一个卡片/区域一个卡片/区域地更新
  1. 保持 HTML 标签结构完全不变（<div class="tab-panel">、<div class="card"> 等标签不动）
  2. 只替换标签内部的文本内容（如卡片标题、正文、数据值）
  3. 可以按区域批量替换，例如：
     - 一个卡片一个卡片替换（替换 <div class="card-body">...</div> 内部内容）
     - 一个概览卡一个概览卡替换
     - 一个市场数据行一个数据行替换
  4. 逐个 Tab 完成，顺序：Tab 0 → Tab 1 → Tab 2 → Tab 3 → Tab 4 → Tab 5 → Tab 6 → Tab 7
  5. 最后统一检查涨红跌绿和文字高亮

错误做法（绝对禁止）：
  - 用 Write 工具一次性写入整个文件 ❌
  - 替换包含 HTML 标签的整个区域（如从 <div class="tab-panel"> 到 </div>）❌
  - 修改/删除/新增任何 HTML 标签 ❌
  - 用脚本批量替换后整体覆写文件 ❌
  - 任何可能导致 CSS/JS/HTML 结构被覆盖的操作 ❌
```

## 二、数据引用规范（铁律）

### 2.1 数据时效性要求

| 数据类型    | 要求             | 示例                     |
| ------- | -------------- | ---------------------- |
| 价格/指数数据 | 必须引用最新可得日期     | "标普5/12收盘7412.84"      |
| 基本面数据   | 可引用最近可得，必须标注来源 | "截至2025年报"             |
| 政策/事件   | 必须引用最新进展       | "美联储5/7决议"             |
| 利率/国债   | 引用最新可得         | "1年定存1.65%（2026年5月数据）" |

### 2.2 禁止行为

- ❌ 严禁编造数据
- ❌ 严禁使用过期数据冒充当日数据
- ❌ 严禁"银行护盘"等无来源说法

### 2.3 数据标注格式

- 价格数据：`"标普5/12收盘7412.84（+0.19%）"`
- 基本面数据：`"截至2025年报：归母净利润3685.62亿元"`
- 未披露数据：`"一季报尚未单独披露（最新数据截至2025年报）"`

**涨跌颜色规则（A股惯例，严格执行）**：

- 🔴 **上涨/利好/正面** → 红色（`#f85149` / `var(--red)` / `class="up"`）
- 🟢 **下跌/利空/负面** → 绿色（`#3fb950` / `var(--green)` / `class="down"`）
- ⚪ **持平/中性** → 灰色（`class="neutral"`）
- 🔵 **中性信息/数据** → 蓝色（`#00d4ff` / `var(--accent)`）
- 🟡 **警示/关注** → 黄色（`#f0b429`）

**常见易错场景**：

- "大涨X%"、"创新高"、"领涨" → 必须用**红色**，不能用绿色
- "大跌X%"、"净卖出"、"暴跌" → 必须用**绿色**，不能用红色
- "偏悲观"、"恐慌"、"谨慎" → 必须用**绿色**（负面=绿色）
- "偏乐观"、"信心"、"看好" → 必须用**红色**（正面=红色）
- "外围市场上涨"相关卡片背景和文字 → 用**红色**系，不能用绿色系
- 涨跌幅数值（如+3.14%、-2.04%）→ 上涨红色、下跌绿色
- 标签（如"今日领涨"）→ 红色背景+红色文字

## 四、格式规范

- **涨红跌绿**：`.up`=红色（涨），`.down`=绿色（跌），`.neutral`=灰色（持平）
- **搜索不到的数据**：标注"暂无数据"，不编造
- **零"待更新"为目标**：亚太5个+大宗4个+汇率4个尽量全部搜齐
- 所有的文字内容需要突出重点（通过颜色突出，不要改字体大小和其他的）

#### Tab 6 · 理财数据参考（重点——之前经常遗漏！）

- **社区热门话题**（10个话题）🔥 **最常遗漏！**
  
  - **标题格式要求**：必须采用**问题/议题形式**，引发讨论，如：
    
    - ✅ "银行股创新高，还能追吗？"
    - ✅ "科技股分化严重，该买龙头还是抄底小票？"
    - ✅ "原油大跌5.8%，是抄底机会还是风险释放？"
    - ❌ "银行股创新高"（太简单，无讨论性）
    - ❌ "原油大跌"（陈述句，非问题）
  
  - **正文格式要求**：必须采用**多方观点讨论**形式，包含：
    
    - 2-3个不同立场的观点（如：乐观派/谨慎派/分析师，或：抄底派/风险派/理性派）
    - 每个观点用不同颜色标识发言方（如：`@价值投资者：`红色，`@机构分析师：`青色，`@谨慎派：`绿色）
    - 每个观点至少1-2句话，包含具体数据或论据
    - 最后以"老盛观点"作为总结建议（内容需要丰富一下，加一些原因分析）
  
  - **示例格式**：
    
    ```
    标题：银行股创新高，还能追吗？
    正文：
    @价值投资者：建设银行股价10.10元创历史新高，农业银行年内涨幅52.66%...
    @机构研究员：险资、北向资金持续加仓银行股...
    @谨慎派：短期涨幅较大，不建议追高...
    老盛观点：银行股创新高是价值发现的体现...
    ```
