## 四、数据准确性检查

### 4.1 股息率 ⭐重点检查
- [ ] 工商银行/农业银行股息率约4.5%（非6.5-6.8%）
- [ ] 招商银行股息率约5.3-7%
- [ ] 区分A股股息率和H股股息率，报告中标注"A股股息率约X%"
- [ ] 禁止编造股息率数据

### 4.2 涨跌颜色（涨红跌绿）
- [ ] A股、港股、美股上涨用红色（.up）
- [ ] A股、港股、美股下跌用绿色（.down）
- [ ] 加密货币、外汇按国际惯例（涨绿跌红）

### 4.3 数据标注
- [ ] 价格/指数数据引用最新可得日期
- [ ] 基本面数据标注来源日期
- [ ] 搜索不到的数据标注"暂无数据"或引用最近可得日期
- [ ] 无旧日期残留（引用前日数据除外）

## 五、原创性检查 ⭐核心检查

### 5.1 检查方法
将当天报告的Tab 4/5/6/7文字与前一天报告逐段对比

### 5.2 判定标准
- [ ] Tab 4/5/6/7 所有卡片标题基于当日事件重新拟定（禁止标题高度相似）
- [ ] Tab 5 每个标的的bullet文字基于最新市场环境重新撰写
- [ ] Tab 6 所有文字板块基于当日市场环境重新撰写
- [ ] Tab 7 所有模块文字基于当日数据重新撰写
- [ ] 无>30字连续相同
- [ ] 无同义词替换+分析逻辑完全相同
- [ ] 低估值板块2个标的与前一天不同（轮换）
- [ ] 分析角度与前一天不同（即使讨论同一板块）
