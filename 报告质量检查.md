# 老盛早知道报告质量检查清单（优化版）

> **核心原则**：先查数据，再查结构，最后查细节。不通过即打回，不得跳过。

---

## 一、数据及时性检查

**检查时机**：所有其他检查之前必须首先执行
**不通过处理**：发现旧数据立即打回，重新搜索替换

### 1.1 关键日期定义

- **报告日期** = T日（如5月21日）
- **数据日期** = T-1日（如5月20日）
- **模板旧日期** = template.html中残留的日期

### 1.2 自动化检测

**检测步骤**：

1. 检查旧日期残留（T-3日及更早，输出应≤3处）
2. 验证新数据已写入（关键数据出现≥10处）
3. 检查数据一致性（同一指标在不同位置应相同）

### 1.3 判定标准

| 检查项   | 通过标准           | 不通过信号   |
| ----- | -------------- | ------- |
| 旧日期残留 | ≤3处（仅允许合理历史回顾） | >3处     |
| 新数据覆盖 | ≥10处关键数据       | <5处     |
| 数据一致性 | 同一指标全报告一致      | 同一指标不同值 |

### 1.4 逐Tab抽查清单

| Tab   | 抽查位置           | 检查内容        |
| ----- | -------------- | ----------- |
| Tab 0 | 4张摘要卡片数据       | 是否为T-1日数据   |
| Tab 1 | 第1张新闻卡片        | 是否为最近1-2天新闻 |
| Tab 3 | A股market-block | 指数是否为T-1日收盘 |
| Tab 5 | 工商银行stock-card | 股价是否为T-1日   |
| Tab 7 | 市场面card-body   | 描述是否为T-1日市场 |

## 三、数据准确性检查

### 3.1 涨跌颜色检查

- [ ] A股/港股/美股：**涨红跌绿**（.up红色，.down绿色）
- [ ] 加密货币/外汇：涨红跌绿（国际惯例）
- [ ] 上涨/利好场景（"大涨"、"创新高"、"领涨"、"偏乐观"、"看好"等）→ 使用**红色**（#f85149 / var(--red) / class="up"）
- [ ] 下跌/利空场景（"大跌"、"净卖出"、"暴跌"、"偏悲观"、"恐慌"、"谨慎"等）→ 使用**绿色**（#3fb950 / var(--green) / class="down"）
- [ ] 外围市场上涨相关卡片（背景、标题、标签、数值）→ 使用**红色**系，不能用绿色系
- [ ] 涨跌幅数值（如+3.14%）→ 上涨红色、下跌绿色
- [ ] 标签语义匹配（如"今日领涨"标签 → 红色背景+红色文字，不能用绿色）
- [ ] 无"高股息"、"稳定现金流"等正面描述误用绿色的情况（注：股息率数值可用绿色表示稳健属性，但"领涨"、"大涨"等涨跌语义必须用红色）

### 3.2 数据标注检查

- [ ] 价格/指数数据标注日期（如"5/20收盘"）
- [ ] 无旧日期残留（引用前日数据除外）

### 3.3 数据完整性检查（⭐核心检查）

**问题描述**：股价、指数等关键数据可能显示为"--"或缺失，必须全部获取真实数据。

**检查方法**：

- [ ] **全局搜索"--"**：报告中不存在任何"--"缺失数据标记
- [ ] **页头ticker数据**：上证指数、深证成指、创业板指、道琼斯等全部已填充
- [ ] **A股市场数据**：上证指数、深证成指、创业板指、科创50、沪深300、成交额、涨跌家数、北向资金全部已填充
- [ ] **港股市场数据**：恒生指数、恒生科技指数、国企指数、南向资金全部已填充
- [ ] **亚太市场数据**：日经225、韩国KOSPI、台湾加权、印度Sensex、澳洲ASX200全部已填充
- [ ] **欧洲市场数据**：英国富时100、德国DAX30、法国CAC40、斯托克50全部已填充
- [ ] **美股市场数据**：道琼斯、标普500、纳斯达克、英伟达、特斯拉、ARM全部已填充
- [ ] **大宗商品数据**：WTI原油、布伦特原油、国际黄金、上海金、白银全部已填充
- [ ] **加密货币数据**：比特币BTC、以太坊ETH、BTC ETF、市场情绪全部已填充
- [ ] **汇率债券数据**：USD/CNY中间价、在岸汇率、美元指数、美10年期、美30年期、中10年期全部已填充
- [ ] **关注标的股价**：工商银行、建设银行、招商银行、中国银行、农业银行、重庆银行、宁波银行、长江电力、大秦铁路、中国神华、中国平安、中国移动、中国核电全部已填充
- [ ] **估值参考数据**：银行板块PB、上证PE、银行股息率、中证红利PE、中海油桶油利润、中国移动PB、招商银行PB、中国核电PE、黄金价格全部已填充

**数据缺失零容忍原则**：

| 数据类别       | 必须获取 | 可接受替代方案     |
| ---------- | ---- | ----------- |
| A股/港股/美股指数 | ✅ 必须 | 无           |
| 个股股价       | ✅ 必须 | 无           |
| 亚太/欧洲指数    | ✅ 必须 | 无           |
| 大宗商品价格     | ✅ 必须 | 无           |
| 汇率数据       | ✅ 必须 | 无           |
| 债券收益率      | ✅ 必须 | 无           |
| 加密货币       | ✅ 必须 | 标注"周末震荡"等状态 |

**合格标准**：

- 缺失数据（"--"）数量 = 0
- 所有"--"必须替换为真实数据或合理状态描述（如"周末休市"需确认是真实状态）
- 不允许以"--"代替未获取的数据

**自动化检查命令**：

```bash
grep -n '"--"' LaoshengDailyBrief_YYYYMMDD.html | wc -l  # 应返回0
grep -n '>--<' LaoshengDailyBrief_YYYYMMDD.html | wc -l  # 应返回0
```
